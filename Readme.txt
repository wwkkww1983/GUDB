类库项目
    -GUDB.Common: 基础公共类库
    -GUDB.Model :基础业务模型
    -GUDB.DAL:数据库访问
    -GUDB.IDAL:数据库访问接口
    -GUDB.BLL :业务层， 业务处理
    -GUDB.IBLL ：业务层接口， 约束业务接口或者服务接口
    -GUDB.Services :服务层, 此层使用WebService或者WCF将业务封装成对外暴露的业务，也就是SOA化
    -GUDB.WorkFlow:工作流层， 放所有工作流相关东西
    -GUDB.UnitTest:单元测试
